declarative-tracking @ Google Analytics

Event and Goal Conversion Tracking

With declarative-tracking library you can use plain HTML attributes to set up event and goal conversion tracking on your landing pages. No JavaScript programming required!

Google Analytics

Outbound Link Tracking

With pure HTML you can send event data to Google Analytics when users click specified links


<a data-track="click"
 data-track-service="google-analytics"
 data-track-event-category="Outbound link"
 data-track-event-action="click"
 data-track-event-label="github.com/arturmoczulski/declarative-tracking"
 href="github.com/arturmoczulski/declarative-tracking">
 Check out GitHub Repo »
</a>
